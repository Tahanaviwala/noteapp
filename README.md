# NoteApp — Flask + React Task/Note Web App

A full-fledged task & note-taking application with Flask REST backend and React frontend.

## Features
- **Tasks**: create, edit, toggle done, delete, filter (all/active/done), priority (low/med/high), due date
- **Notes**: create, edit, delete, pin, tag, live search, responsive grid
- Persistent SQLite storage
- REST API with CORS
- Dark themed UI

## Deployment

Deployed to Vercel. `/api/*` routes to Flask serverless, everything else to the React static build.
